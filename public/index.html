Welcome to the HP-API!

Use these routes to request data on Harry Potter characters:

  • All Characters
  • Specific Character by ID
  • Hogwarts Students
  • Hogwarts Staff
  • Characters in a House

No API key is required. Feel free to explore and enjoy the magic!
